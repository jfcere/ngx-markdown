Remote Url

Using component with `src` property to fetch remote markdown file `app/remote/demo.md`
Using component with static `python` code block {{ demoPython | language: 'python' }}
Using directive with `src` property to fetch remote html file `app/remote/demo.html`

Using directive with `src` property to fetch remote C++ file `app/remote/demo.cpp`

Variable Binding

Using component or directive with `data` property allow to bind a variable that will update the DOM when value changes

Using `language` pipe you can specify the language of the variable content for synthax highlights

Pipe Usage

Using `markdown` pipe to transform markdown to HTML allow you to chain pipe transformations and will update the DOM when value changes
In the example below, `pipeMarkdown` property does not contains any `back-ticks` to set the content language but will be chain with `language` pipe instead to specify synthax highlights language along with `markdown` pipe for conversion
